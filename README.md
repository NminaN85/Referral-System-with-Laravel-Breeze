
## # Referral System with Laravel & Breeze

This project demonstrates how to build a **Referral System** using **Laravel** and **Breeze** for user authentication. 
This system allows users to refer other users using a unique referral code, and track referral signups in real-time.

## Features:
- User Registration & Authentication using **Laravel Breeze**.
- Ability to create and manage unique referral codes.
- Track referral users who sign up using the referral code..

## Technologies Used:
- **Laravel** for backend.
- **Breeze** for authentication and simple UI.
- **MySQL** database to store referral data.
- **Blade** templating engine.
